# backups\

Timestamped backups of Hayeong's state and important files.

## What Lives Here

Periodic snapshots of:
- memory.json
- identity.json
- core.json shared state
- Knowledge files

## What To Know

Before any significant update to Brain\ or main.py, a backup should be
created here. Filename format: `[YYYY-MM-DD_HHMM]_[description]`

Hayeong can write her own backups here before attempting self-updates.
